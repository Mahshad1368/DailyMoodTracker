## After Generating Icons

### Step 1: Download All 5 Icons
Download each generated 1024x1024px image:
- `happy-1024.png`
- `neutral-1024.png`
- `sad-1024.png`
- `angry-1024.png`
- `sleepy-1024.png`

### Step 2: Resize to Required Sizes

You need to create @2x and @3x versions of each icon.

#### Option A: Use Online Tool
**ImageOptim** or **Squoosh.app**:
1. Go to https://squoosh.app
2. Upload 1024x1024 image
3. Resize to 120x120 ‚Üí Save as `AppIcon-Happy@2x.png`
4. Resize to 180x180 ‚Üí Save as `AppIcon-Happy@3x.png`
5. Repeat for all 5 moods

#### Option B: Use macOS Preview
1. Open image in Preview
2. Tools ‚Üí Adjust Size...
3. Change to 120 x 120 pixels
4. File ‚Üí Export ‚Üí Save as `AppIcon-Happy@2x.png`
5. Repeat with 180 x 180 for @3x version

#### Option C: Use Command Line (macOS/Linux)
```bash
# Install ImageMagick first: brew install imagemagick

# Happy icons
convert happy-1024.png -resize 120x120 AppIcon-Happy@2x.png
convert happy-1024.png -resize 180x180 AppIcon-Happy@3x.png

# Neutral icons
convert neutral-1024.png -resize 120x120 AppIcon-Neutral@2x.png
convert neutral-1024.png -resize 180x180 AppIcon-Neutral@3x.png

# Sad icons
convert sad-1024.png -resize 120x120 AppIcon-Sad@2x.png
convert sad-1024.png -resize 180x180 AppIcon-Sad@3x.png

# Angry icons
convert angry-1024.png -resize 120x120 AppIcon-Angry@2x.png
convert angry-1024.png -resize 180x180 AppIcon-Angry@3x.png

# Sleepy icons
convert sleepy-1024.png -resize 120x120 AppIcon-Sleepy@2x.png
convert sleepy-1024.png -resize 180x180 AppIcon-Sleepy@3x.png
```

## Troubleshooting AI Generation

### Problem: Emoji looks wrong or distorted
**Solution**: Add to prompt: "use standard iOS Apple emoji style, not 3D rendered"

### Problem: Background has patterns/textures
**Solution**: Add to prompt: "flat solid gradient background, no patterns, no textures"

### Problem: Text appears in image
**Solution**: Add to prompt: "absolutely no text, no labels, no words"

### Problem: Icon has rounded corners already
**Solution**: That's okay! iOS applies its own mask. The generated icon should be square.

### Problem: Colors don't match exactly
**Solution**: Try these additions:
- "exact hex colors #FFD93D to #FFAA80"
- "precise color gradient from yellow #FFD93D to peach #FFAA80"
